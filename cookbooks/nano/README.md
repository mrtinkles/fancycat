Description
===========

Installs nano.

Requirements
============

## Platform:

* Ubuntu/Debian
* Red Hat/CentOS/Fedora/Scientific
* ArchLinux

Attributes
==========

None.

Usage
=====

Put recipe[nano] in a run list, or include_recipe 'nano' to ensure that nano is installed on your systems.
